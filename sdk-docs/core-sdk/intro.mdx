---
title: "Intro"
description: FunKit empowers you to create feature-rich and extensible smart wallets built on account abstraction. Leveraging FunKit, you can customize gas behavior, adopt multi-sig and common authentication methods, monetize your application, execute any transactions from smart wallets, and much more.
---

These guides only cover the **FunKit Core SDK** which does not provide any frontend-specific optimizations. Check out our React SDK if you want to further simplify your application with our React hooks.

## Installation

```bash
npm i @funkit/core --save
# or
yarn add @funkit/core
```

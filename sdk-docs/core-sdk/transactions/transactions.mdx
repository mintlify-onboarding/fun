title: "Transactions"
sidebarTitle: Overview
description: With the FunKit SDK, you can execute any transaction interpretable by an EVM blockchain. You can execute these transactions on any supported chain without switching providers. The SDK also comes with a set of pre-built methods for the most popular transactions. Each of these first class actions support gas estimation and custom payment options.
---

High-level interfaces are provided for the most common actions including transfers, token swaps, and limit order placement. Lower-level support is available for performing any arbitrary smart contract operations.

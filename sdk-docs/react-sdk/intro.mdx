title: "Intro"
description: FunKit empowers you to create feature-rich and extensible smart wallets built on account abstraction. Leveraging the FunKit, you can customize gas behavior, adopt multi-sig and common authentication method, monetize your application, execute any transactions from smart wallets, and much more.
---

These guides only cover the **FunKit React SDK** which is designed to implement high-level functions and React hooks to simplify frontend integrations with FunKit. See the Core SDK docs for information on the lower-level SDK interfaces.

## Installation

```bash
npm i @funkit/react --save
# or
yarn add @funkit/react
```
